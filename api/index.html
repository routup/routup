API Reference ​

Routup provides tree-shakeable helper functions for working with requests and responses. All helpers take an IRoutupEvent as their first argument.

  • Request Helpers — Read headers, body, client info, content negotiation, and metadata
  • Response Helpers — Send responses, set headers, manage caching, and stream events
